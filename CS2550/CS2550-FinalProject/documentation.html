Calendar Implementation

Table: The table was created using html code that was generated by javascript in view.js, using functions from model.js

Arrays: 2D arrays contain the proper dates for each corresponding cell.

Dates:Actual dates are generated by creating a new "DATE" object. Using methods of that object, I get the Month, Year, and first weekday of the month.

2D Array insertion: A for loop in getCorrectDateInfo() is used to update the grid with dates

2D array update when changing month: A near identical for loop is used to update the array and the corresponding grid cells using the updatedGetCorrectDateInfo() function

Selected Cell: A simple function updatedSelectedCell(row,column) updates the innerHTML of the element showing the coordinates on the Calendar.

On click event creation: When cells are first generated by javascript, they are dynamically assigned an ID name according to it's [row][column] position. This allows me to create a div container on top of the cell that has been clicked on(A click returns the cell ID, which is [row][column]). A variable is created, its content is html code that creates a textfield and button for creating events. That variable is then assigned as the innerHTML of the container div.I append the div to the cell

Changing Months A select element is generated by javascript, the options are months of the year. Onclick, they activate the updateMonth() function.

updateMonth(): This assigns the correct corresponding dates with the cell according to the chosen month(You can compare to a real calendar, it is correct). After than, it calls the function requestCalendarJson()

requestCalendarJson(): This function is responsible for XMLHttprequests. It pulls info from the sample.json file and updates the grid with the events parsed form the JSON file

sample.json: The all encompasing object in this file is the year "2019". Properties of "2019" are months of the year("January, February, March, etc"). The property within each month is a 2D array, which is parsed and used to update the original 2D array used to generate the calendar in the first place.
