SOFTWARE DESIGN & DESCRIPTION

The type of project I will be working on will be an 'Event Calendar'


This 'Event Calendar' will show the current month , including the days of the week, in a grid. Each square within the grid will show any events for that day. Each square will be interactive, allowing you to view the information in greater detail. It will have the ability to create new events.

Model


VARIABLES


month- the current calendar month

year- the current calendar year

day- the current calendar day

weekday- the current calendar weekday

grid- an array reflecting the day of year for each cell

eventsGrid- an array containing events for each day


FUNCTIONS
(These functions are called in "./Scripts/view.js" in function "generateCalendarBody" at line 75 and 76 )


getCellDate(row,col) - returns an item from a 2D array. This item is the day of the year for the given cell

getCellEvents(row,col) - returns an item from a 2d array. These items are the events for a given day.
